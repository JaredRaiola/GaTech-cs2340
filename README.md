# RYSK

A web application for playing RYSK, a spin-off of RISK made by Team 17.

## How to run

Start the Play app:

```bash
sbt run
```

And open [http://localhost:9000/](http://localhost:9000/)

Documentation and project blueprinting are in "root/designProcess"
